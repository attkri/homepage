## KI

### [Anthropic launcht Claude Code Security -- Cybersecurity-Aktien verlieren](https://www.heise.de/news/Anthropic-launcht-Claude-Code-Security-Cybersecurity-Aktien-verlieren-11185198.html)

Anthropic hat "Claude Code Security" vorgestellt -- ein KI-gestütztes Tool, das Sicherheitslücken in Code automatisch erkennen und beheben soll. Die Reaktion der Börse war unmittelbar: Aktien klassischer Cybersecurity-Anbieter gaben spürbar nach. Marktbeobachter diskutieren, ob KI-Sicherheitstools traditionelle Anbieter mittelfristig verdrängen.

*Quelle: heise.de · 21. Feb 2026*

### [Meta investiert Milliarden in AMD-GPUs mit Option auf Firmenanteile](https://www.heise.de/news/Auf-OpenAI-folgt-Meta-AMD-verkauft-GPUs-fuer-Milliarden-von-US-Dollar-11188239.html)

Meta kauft AMD-GPUs im Milliardenmaßstab und erhält im Gegenzug die Option auf AMD-Unternehmensanteile. Nach einem ähnlichen Deal zwischen OpenAI und AMD signalisiert auch Meta, dass der KI-Infrastruktur-Wettlauf in eine neue Phase eintritt -- weg von Nvidia-Abhängigkeit, hin zu AMD.

*Quelle: heise.de · 24. Feb 2026*

### [Mozilla Firefox 148: Zentraler KI-Kill-Switch für alle KI-Features](https://www.heise.de/news/Firefox-148-bringt-zentrale-KI-Kontrolle-und-neue-Web-Standards-11188046.html)

Mozilla führt in Firefox 148 einen einzigen Schalter ein, mit dem Nutzer alle KI-Funktionen des Browsers zentral deaktivieren können. Damit reagiert Mozilla auf wachsende Datenschutzbedenken und positioniert Firefox als bewusste Alternative zu KI-tiefen Browsern. Ebenfalls an Bord: neue Web-Standards für bessere Interoperabilität.

*Quelle: heise.de · 24. Feb 2026*

## Tools & Dev

### [Visual Studio 2026 Version 18.3.2 erschienen](https://learn.microsoft.com/de-de/visualstudio/releases/2026/release-notes)

Microsoft veröffentlichte Visual Studio 2026 in Version 18.3.2. Im Fokus stehen Fehlerkorrekturen im MSVC-Compiler-Optimizer, der unter bestimmten Bedingungen falschen Code erzeugte -- ein stiller, schwer zu findender Fehler, der C++- und .NET-Entwickler direkt betrifft. Update dringend empfohlen.

*Quelle: learn.microsoft.com · 24. Feb 2026*

### [Outlook New: Microsoft verlängert Opt-out-Phase für Unternehmen bis 2027](https://borncity.com/blog/2026/02/24/outlook-new-microsoft-verlaengert-opt-out-phase-fuer-firmen-bis-2027/)

Eigentlich sollte die Möglichkeit, Outlook New zugunsten von Outlook Classic zurückzustellen, im April 2026 enden. Microsoft verlängert diese Opt-out-Phase nun bis 2027. Unternehmen, die auf Outlook Classic angewiesen sind, gewinnen damit wertvolle Zeit für die Migration.

*Quelle: borncity.com · 24. Feb 2026*

## IT-Security

### [Google Chrome 145: Update schließt drei Schwachstellen inkl. CVE-2026-2441](https://borncity.com/blog/2026/02/24/google-chrome-145-0-7632-116-117-fixt-schwachstellen/)

Google veröffentlichte Chrome 145.0.7632.116/117 mit Korrekturen für drei Sicherheitslücken. Im Mittelpunkt steht CVE-2026-2441, eine Use-after-free-Schwachstelle mit hohem Schweregrad, die auch alle Chromium-basierten Browser (Microsoft Edge, Brave, Opera) betrifft. Sofort aktualisieren.

*Quelle: borncity.com · 24. Feb 2026*

### [Datenleak bei Safran Group: Daten des französischen Rüstungskonzerns im Netz](https://borncity.com/blog/2026/02/23/daten-der-safran-group-franz-ruestungskonzern-geleakt/)

Ein Angreifer behauptet, Daten aus den IT-Systemen von Safran -- Hersteller von Triebwerken und Avionik für zivile und militärische Luftfahrt -- entwendet zu haben. Erste Datensätze wurden öffentlich als Beleg veröffentlicht. Der Vorfall hat mögliche geopolitische Implikationen.

*Quelle: borncity.com · 23. Feb 2026*

### [Windows Notepad: Details zu CVE-2026-20841 veröffentlicht](https://borncity.com/blog/2026/02/23/windows-notepad-cve-2026-20841-und-bald-grafiken-einbindbar/)

Die Zero-Day-Initiative hat technische Details zur Notepad-Schwachstelle CVE-2026-20841 nachgeliefert, die mit den Windows-Updates vom 10. Februar 2026 geschlossen wurde. Außerdem zeigt Microsoft in einem Preview, dass Notepad künftig Grafiken einbinden können soll.

*Quelle: borncity.com · 23. Feb 2026*
