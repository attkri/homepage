## Breaking News

### Topic-Cluster: Microsoft macht aus einzelnen Assistenten koordinierte KI-Teams und drückt Copilot damit tiefer in reale Unternehmensarbeit

Microsoft hebt Copilot Studio mit Multi-Agenten-Orchestrierung, Evaluierung und Governance auf die nächste Stufe. Mehrere spezialisierte Agenten sollen Aufgaben untereinander delegieren, gemeinsam auf Unternehmensdaten zugreifen und in regulierten Umgebungen kontrollierbar bleiben. Zusammen mit neuen Partnerschaften wie Kyndryl wird klar, wohin die Reise geht: weniger Einzel-Chatbots, mehr arbeitsteilige KI-Systeme für echte Betriebsabläufe.

## Tools & Dev

### Topic-Cluster: GitHub baut Copilot zum echten Cloud-Agenten aus und zieht Governance gleich mit

GitHub öffnet Copilot Cloud Agent jetzt auch für Research-, Planungs- und Branch-Workflows ohne sofortigen Pull Request. Dazu kommen organisationweite Custom Instructions als GA und ein deutlich kräftigeres Copilot in Visual Studio mit eigenen Agents, Skills, MCP-Governance und Profiling-Unterstützung. Das ergibt zusammen keinen kleinen Produkt-Refresh, sondern einen sichtbaren Schritt Richtung standardisierter Agentenarbeit im Dev-Alltag.

## IT-Security

### [Microsoft warnt vor neuer WhatsApp-Malware für Windows](https://borncity.com/news/microsoft-warnt-vor-neuer-whatsapp-malware-fuer-windows/)

Die Kampagne kombiniert Social Engineering über kompromittierte Kontakte mit LOTL-Techniken auf Windows und macht damit genau den Bereich gefährlich, in dem Mitarbeitende gern „nur kurz“ einen Anhang öffnen. Für Unternehmen ist das besonders heikel, weil der Angriffsweg über einen vertrauten Messenger läuft, der im Alltag oft weniger kritisch wahrgenommen wird als E-Mail.
